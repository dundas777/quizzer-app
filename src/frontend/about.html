About Quizzer

Quizzer is a multiple choice quiz website where users can test their knowledge across various categories. Each quiz presents a series of questions with several possible answers, allowing users to select the correct one and track their score.

This website is specifically constructed to help practice test automation, both for UI and API testing. It provides a realistic environment for automating user interactions and validating backend API responses.

The site contains a variety of HTML elements, such as buttons, radio buttons, select dropdowns, and dynamic content. These require different locator strategies, making it ideal for learning and practicing robust automation techniques.
